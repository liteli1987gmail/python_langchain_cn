# HNSWLib

:::info 兼容性
仅适用于Node.js。
:::

HNSWLib是一个内存向量存储器，可以保存到文件中。它使用[HNSWLib](https://github.com/nmslib/hnswlib)。

## 设置

:::注意

 **在Windows上**，你可能需要先安装[Visual Studio](https://visualstudio.microsoft.com/downloads/)才能正确构建`hnswlib-node`包。

:::

您可以通过以下方式进行安装

```bash npm2yarn
npm install hnswlib-node

```
