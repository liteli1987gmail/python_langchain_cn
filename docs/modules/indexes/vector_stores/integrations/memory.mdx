# `MemoryVectorStore`

MemoryVectorStore是一个内存中的暂存向量存储器，用于在内存中存储嵌入，并做精确的线性搜索以找到最相似的嵌入。默认的相似度度量是余弦相似度，但可以更改为[ml-distance](https://mljs.github.io/distance/modules/similarity.html)支持的任何相似度度量方式。
