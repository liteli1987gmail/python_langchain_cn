# Chroma（嵌入式的开源Apache 2.0数据库)


Chroma是一个开源的Apache 2.0嵌入式数据库。


## 设置


1. 在计算机上使用Docker运行Chroma [文档](https://docs.trychroma.com/api-reference)
2. 安装Chroma JS SDK。


```bash npm2yarn
npm install -S chromadb

```
