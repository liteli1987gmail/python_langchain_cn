# 召回器（Retrievers)

:::info
[概念指南](https://docs.langchain.com/docs/components/indexing/retriever)
:::

一种存储数据的方式，可以通过语言模型进行查询。这个对象必须公开的唯一接口是一个 `getRelevantDocuments` 方法，该方法接受一个字符串查询并返回一个文档列表。
