# Notion markdown export

本示例介绍如何从导出的 Notion 页面中加载数据。 

首先，按照官方说明 [这里](https://www.notion.so/help/export-your-content) 导出 Notion 页面为 **Markdown & CSV**。确保选择 `包括子页面` 和 `为子页面创建文件夹`。

然后，解压下载的文件并将未压缩的文件夹移动到存储库中。它应该包含你页面的 markdown 文件。

一旦文件夹在存储库中，只需运行下面的示例即可:
