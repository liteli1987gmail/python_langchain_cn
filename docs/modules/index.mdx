# 模块（Modules）

![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


LangChain为以下模块提供标准、可扩展的接口和外部集成，按复杂程度从低到高列出:

#### [模型输入输出](/docs/modules/model_io/)
与语言模型进行接口交互
#### [数据连接](/docs/modules/data_connection/)
与特定应用程序数据进行接口交互
#### [链式调用](/docs/modules/chains/)
构造调用序列
#### [代理](/docs/modules/agents/)
根据高级指令让链式调用选择使用哪些工具
#### [内存](/docs/modules/memory/)
在链式调用的运行之间保持应用程序状态
#### [回调函数](/docs/modules/callbacks/)
记录和流式传输任何链式调用的中间步骤
