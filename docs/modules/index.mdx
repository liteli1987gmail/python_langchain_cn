# 模块

LangChain为以下模块提供标准、可扩展的接口和外部集成，按复杂性从低到高列出:

#### [模型 I/O](/docs/modules/model_io/)
与语言模型进行交互
#### [检索](/docs/modules/data_connection/)
与特定应用程序数据进行交互
#### [链式调用](/docs/modules/chains/)
构建调用序列
#### [代理](/docs/modules/agents/)
根据高级指令选择使用哪些工具
#### [内存](/docs/modules/memory/)
在链的运行之间持久化应用程序状态
#### [回调函数](/docs/modules/callbacks/)
记录和传输任何链的中间步骤
