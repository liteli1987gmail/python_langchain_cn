# 组合

![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


本笔记本介绍了如何将多个提示组合在一起。当您想要重用提示的部分时，这将非常有用。这可以通过PipelinePrompt来实现。PipelinePrompt由两个主要部分组成:

- 最终提示: 返回的最终提示
- 管道提示: 由一个字符串名称和一个提示模板组成的元组列表。每个提示模板将被格式化，然后作为相同名称的变量传递给未来的提示模板。
