# 提示模板

![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


提示模板是生成语言模型提示的预定义配方。

模板可能包括指令、少量示例以及适用于特定任务的特定上下文和问题。

LangChain提供了创建和使用提示模板的工具。

LangChain致力于创建与模型无关的模板，以便在不同的语言模型之间轻松重用现有模板。
