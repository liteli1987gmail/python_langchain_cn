# 提示(Prompts)
![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


编写模型的新方式是通过提示。
一个 **提示(prompt)** 指的是输入模型的内容。
这个输入通常由多个组件构成。
LangChain 提供了多个类和函数，使构建和处理提示变得简单。

- [提示模板(Prompt templates)](/docs/modules/model_io/prompts/prompt_templates/): 为模型输入添加参数
- [示例选择器(Example selectors)](/docs/modules/model_io/prompts/example_selectors/): 动态选择在提示中包含的示例
