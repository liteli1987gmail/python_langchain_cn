# LLMs

![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


:::info
前往[集成](/docs/integrations/llms/)以获取与LLM提供商的内置集成的文档。
:::

大型语言模型（LLMs）是LangChain的核心组件。LangChain不提供自己的LLMs，而是提供与许多不同LLMs交互的标准接口。

## 入门

有很多LLM提供商（OpenAI、Cohere、Hugging Face等）- `LLM`类旨在为所有这些提供商提供标准接口。

在本教程中，我们将使用OpenAI LLM包装器，尽管强调的功能对于所有LLM类型都是通用的。
