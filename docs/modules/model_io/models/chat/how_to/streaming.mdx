# 实时流媒体 streaming
![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


一些聊天模型提供实时流媒体响应。这意味着您无需等待完整响应返回，而是可以在其可用时开始处理响应。如果您希望在生成响应时将其显示给用户，或者希望在生成响应时处理响应，这将非常有用。
