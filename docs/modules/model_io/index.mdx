# 模型输入输出 ( Model I/O )

任何语言模型应用的核心元素是...模型的输入和输出。LangChain为您提供了与任何语言模型进行接口交互的基本组件。

- [提示](/docs/modules/model_io/prompts/): 将模型输入模板化、动态选择和管理
- [语言模型](/docs/modules/model_io/models/): 通过常见接口调用语言模型
- [输出解析器](/docs/modules/model_io/output_parsers/): 从模型输出中提取信息
