# 摘要 summarize

![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')

摘要链可以用于对多个文档进行摘要。一种方法是将多个较小的文档输入，将它们划分为块，并使用MapReduceDocumentsChain对其进行操作。您还可以选择将摘要链设置为StuffDocumentsChain或RefineDocumentsChain。
