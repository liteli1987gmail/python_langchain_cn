# 链 (Chains)

在独立使用LLM进行简单应用时是可以的，
但更复杂的应用需要将LLM进行链式调用 - 可以是与其他LLM进行链式调用，也可以是与其他组件进行链式调用。

LangChain为这种“链式”应用提供了**Chain**接口。我们将Chain定义得非常通用，作为一个调用组件的序列，其中可以包含其他链。基本接口很简单:

## 为什么我们需要链? (Why do we need chains?)

链允许我们将多个组件组合在一起，创建一个单一而连贯的应用程序。例如，我们可以创建一个链，接受用户输入，使用PromptTemplate进行格式化，然后将格式化的响应传递给LLM。我们可以通过将多个链组合在一起或将链与其他组件组合来构建更复杂的链。
