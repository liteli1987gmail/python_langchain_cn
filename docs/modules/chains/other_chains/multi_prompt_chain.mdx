## `MultiPromptChain`多次提示链


MultiPromptChain 允许 LLM 从多个提示中进行选择。通过提供一组模板/提示以及它们对应的名称和描述来构建该链。该链接受一个字符串作为输入，选择一个合适的提示，然后将输入传递到所选的提示中。
