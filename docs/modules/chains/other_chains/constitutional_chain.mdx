# `宪法链`

`宪法链`是一种链式结构，它确保语言模型的输出遵循预定义的宪法原则。通过纳入特定的规则和指南，`宪法链`可以过滤和修改生成的内容以符合这些原则，从而提供更加受控、道德和上下文恰当的响应。这种机制有助于保持输出的完整性，同时最大程度地减少生成可能违反指南、具有冒犯性或偏离所需上下文的内容的风险。
