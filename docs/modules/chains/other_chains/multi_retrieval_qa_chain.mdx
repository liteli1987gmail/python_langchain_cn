# `MultiRetrievalQAChain`多重检索问题解答链
换行
MultiRetrievalQAChain使LLM能够从多个检索器中进行选择。通过提供一组向量存储器（作为检索器)及其相应的名称和描述来构建链。该链将查询作为输入，选择适当的检索器，并随后将输入馈送到所选的检索器中。
