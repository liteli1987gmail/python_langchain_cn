# 东西文档链（ Stuff documents ）
![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


东西文档链（"东西" 的意思是 "填充" 或 "填充"）是最直接的文档链之一。它接受一个文档列表，将它们全部插入到提示中，并将该提示传递给 LLM。

此链适用于文档较小且大多数调用仅传递少数文档的应用程序。
