# LLM

![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')

LLMChain 是围绕语言模型添加一些功能的简单链。它广泛用于 LangChain 中，包括其他链和代理程序。

LLMChain 由 PromptTemplate 和语言模型（LLM 或聊天模型）组成。它使用提供的输入键值（如果有的话，还包括内存键值）格式化提示模板，将格式化的字符串传递给 LLM 并返回 LLM 输出。
