# LLM

![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')

`LLMChain`是一个简单的链式结构，它在语言模型周围添加了一些功能。它广泛用于LangChain中，包括其他链式结构和代理程序。

`LLMChain`由`PromptTemplate`和语言模型（LLM或聊天模型）组成。它使用提供的输入键值（如果有的话，还包括内存键值）格式化提示模板，将格式化的字符串传递给LLM并返回LLM输出。
