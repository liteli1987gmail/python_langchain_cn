# 数据连接 (Data connection)

许多LLM应用程序需要用户特定的数据，这些数据不属于模型的训练集。LangChain为您提供了加载、转换和查询数据的基础构件:
- [文档加载器 (Document loaders)](/docs/modules/data_connection/document_loaders/): 从许多不同的源加载文档

- [Document loaders](/docs/modules/data_connection/document_loaders/): Load documents from many different sources
- [文档转换器 (Document transformers)](/docs/modules/data_connection/document_transformers/): 分割文档、删除冗余文档等
- [文本嵌入模型 (Text embedding models)](/docs/modules/data_connection/text_embedding/): 将非结构化文本转换为浮点数列表
- [向量存储 (Vector stores)](/docs/modules/data_connection/vectorstores/): 存储和检索嵌入数据
- [检索器 (Retrievers)](/docs/modules/data_connection/retrievers/): 查询您的数据
