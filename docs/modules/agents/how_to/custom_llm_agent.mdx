# 自定义 LLM 代理 custom_llm_agent

![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


本笔记本介绍如何创建您自己的自定义 LLM 代理。

一个 LLM 代理由三个部分组成：

- PromptTemplate：这是可以用来指导语言模型做什么的提示模板
- LLM：这是驱动代理的语言模型
- `stop` 序列：指示 LLM 在找到此字符串时停止生成
- OutputParser：确定如何将 LLMOutput 解析为 AgentAction 或 AgentFinish 对象
