# 会话缓冲窗口记忆 ( Conversation buffer window memory )
![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')


`ConversationBufferWindowMemory` 会随着时间记录会话的交互列表。它只使用最后 K 个交互。这对于保持最近交互的滑动窗口很有用，以避免缓冲区过大
