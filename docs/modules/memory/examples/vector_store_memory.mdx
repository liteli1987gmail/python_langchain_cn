# 基于向量库的内存

`VectorStoreRetrieverMemory`将记忆存储在向量数据库中，并在每次调用时查询最“显著”的前K个文档。

这与大多数其他内存类不同，它没有显式跟踪交互的顺序。

在这种情况下，“文档”是先前的会话片段。这可以用来提到AI之前在对话中被告知的相关信息。
