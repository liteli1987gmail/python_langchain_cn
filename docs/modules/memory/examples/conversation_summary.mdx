# Conversation Summary Memory（对话总结记忆)


对话总结记忆会在对话进行时对其进行总结并储存在记忆中。这个记忆能够被用于将当前对话总结注入到提示/链中。这个记忆在对较长的对话进行总结时非常有用，因为直接在提示中保留所有过去的对话历史信息将会占用过多的token。
