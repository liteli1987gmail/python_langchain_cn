# 摘要 summarize
![LangChain](https://pica.zhimg.com/50/v2-56e8bbb52aa271012541c1fe1ceb11a2_r.gif 'LangChain中文网')

摘要是指对多个较长文档进行概括总结。
这对于将长文档浓缩为核心信息非常有用。

使用摘要链的推荐方法是：

```python
from langchain.chains.summarize import load_summarize_chain
chain = load_summarize_chain(llm, chain_type="map_reduce")
chain.run(docs)
```
