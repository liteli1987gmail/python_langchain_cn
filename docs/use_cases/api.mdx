# 与API交互

:::info
[概念指南](https://docs.langchain.com/docs/use-cases/apis)
:::

许多数据和信息存储在API后面。
本页涵盖了在LangChain中使用API的所有资源。

## 链

如果您刚开始并且有相对简单的API，请从链开始。
链是一系列预定步骤，因此它们很适合入门，因为它们能够给您更多控制，让您
更好地了解发生的情况。

- [API链](../modules/chains/other_chains/api_chain.mdx)

## 代理

代理更复杂，并涉及对LLM的多个查询，以了解要做什么。
代理的缺点是您的控制力少了。优点是它们更强大
这使您可以在更大，更复杂的模式上使用它们。

- [OpenAPI Agent](../modules/agents/toolkits/openapi.md)
