## 链

如果您刚刚开始，并且您有相对简单的API，那么您应该从链开始。
链是一系列预定步骤，因此它们很适合入门，因为它们给您更多控制权并让您更好地了解发生的情况。


- [API链](/docs/modules/chains/how_to/api.html)

## 代理

代理更复杂，涉及到多个查询LML来了解该做什么。
代理的缺点是您的控制权较少。优点是它们更强大，
这使您能够在更大更复杂的架构上使用它们。

- [OpenAPI代理](/docs/modules/agents/tools/toolkits/examples/openapi.html)
