# 个人助理


:::info

[概念指南](https://docs.langchain.com/docs/use-cases/personal-assistants)
:::



我们在这里非常广义地使用“个人助理”这个词。
个人助理具有一些特征:


- 它们可以与外界互动
- 它们具有您的数据知识
- 它们记得您的互动


实际上，LangChain 中的所有功能都与构建个人助理相关。
Highlighting specific parts:



- [Agent 文档](../modules/agents/index.mdx)（用于与外界互动)
- [Index 文档](../modules/indexes/index.mdx)（用于提供数据知识)
- [Memory](../modules/memory/index.mdx) (for helping them remember interactions)
