# 现代财务

>[现代财务](https://www.moderntreasury.com/)简化了复杂的支付操作。它是一个统一的平台，用于支持移动资金的产品和流程。
>- 连接银行和支付系统
>- 实时跟踪交易和余额
>- 自动化可扩展的支付操作

## 安装和设置

它不需要任何特殊设置。
