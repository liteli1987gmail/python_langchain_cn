# StochasticAI

本页面介绍如何在 LangChain 中使用 StochasticAI 生态系统。
它分为两个部分：安装和设置，以及对特定 StochasticAI 包装器的引用。

## 安装和设置
- 使用 `pip install stochasticx` 进行安装
- 获取一个 StochasticAI api 密钥，并将其设置为环境变量（`STOCHASTICAI_API_KEY`）

## 包装器

### LLM

存在一个 StochasticAI LLM 包装器，您可以使用
```python
from langchain.llms import StochasticAI
```
