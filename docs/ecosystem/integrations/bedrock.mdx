# Bedrock

>[Amazon Bedrock](https://aws.amazon.com/bedrock/) 是一个完全托管的服务，通过 API 提供来自领先 AI 创业公司和亚马逊的 FMs，因此您可以从各种 FMs 中选择最适合您的用例的模型。

## 安装和设置

```bash
pip install boto3
```

## LLM

查看一个[使用示例](/docs/modules/model_io/models/llms/integrations/bedrock.html)。

```python
from langchain import Bedrock
```

查看一个[使用示例](/docs/modules/data_connection/text_embedding/integrations/bedrock.html)。

from langchain.embeddings import BedrockEmbeddings
