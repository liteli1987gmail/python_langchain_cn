# Psychic

[Psychic](https://www.psychic.dev/)是一个平台，通过OAuth与`Notion`、`Zendesk`、`Confluence`和`Google Drive`等SaaS工具进行集成，并将这些应用程序的文档同步到SQL或向量数据库中。您可以将其视为非结构化数据的Plaid。

## 安装和设置

```bash
pip install psychicapi
```

Psychic很容易设置-您导入`react`库并使用您从[Psychic仪表板](https://dashboard.psychic.dev/)获得的`Sidekick API`密钥进行配置。当您连接应用程序时，您可以从仪表板中查看这些连接，并使用服务器端库检索数据。

1. 在[仪表板](https://dashboard.psychic.dev/)中创建帐户。
2. 使用[react库](https://docs.psychic.dev/sidekick-link)将Psychic链接模态框添加到前端React应用程序中。您将使用此功能连接SaaS应用程序。
3. 创建连接后，可以按照[示例笔记本](/docs/modules/data_connection/document_loaders/integrations/psychic.html)使用`PsychicLoader`。

## 与其他文档加载器的优势

1. **通用API：** 您只需集成Psychic一次，即可使用我们的通用API检索数据，而无需构建OAuth流程和学习每个SaaS应用的API。
2. **数据同步：** 您的客户SaaS应用中的数据可能很快过时。使用Psychic，您可以配置Webhooks以每天或实时基础上保持文档的最新状态。
3. **简化的OAuth：** Psychic完全处理OAuth，因此您无需花费时间为每个集成创建OAuth客户端、保持访问令牌的有效性并处理OAuth重定向逻辑。
