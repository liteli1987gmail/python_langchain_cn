# ModelScope

本页面介绍如何在LangChain中使用modelscope生态系统。
它分为两个部分：安装和设置，以及特定modelscope包装器的参考。

## 安装和设置

* 使用 `pip install modelscope` 安装Python SDK

## 包装器

### 嵌入

存在一个modelscope嵌入包装器，您可以通过以下方式访问

```python
from langchain.embeddings import ModelScopeEmbeddings
```

有关更详细的步骤，请参阅[此笔记本](/docs/modules/data_connection/text_embedding/integrations/modelscope_hub.html)
