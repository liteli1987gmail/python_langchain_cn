# ModelScope

本页面介绍如何在 LangChain 中使用 modelscope 生态系统。
它分为两个部分：安装和设置，以及特定 modelscope 包装器的参考。

## 安装和设置

* 使用 `pip install modelscope` 安装 Python SDK

## 包装器

### 嵌入

存在一个 modelscope 嵌入包装器，您可以通过以下方式访问

```python
from langchain.embeddings import ModelScopeEmbeddings
```

有关更详细的步骤，请参阅 [此笔记本](/docs/modules/data_connection/text_embedding/integrations/modelscope_hub.html)
