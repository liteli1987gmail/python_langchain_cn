# Microsoft OneDrive

>[微软 OneDrive](https://en.wikipedia.org/wiki/OneDrive) （原名 `SkyDrive` ）是由微软运营的文件托管服务。

## 安装和设置

首先，您需要安装一个Python包。

```bash
pip install o365
```

然后按照[这里的说明](/docs/modules/data_connection/document_loaders/integrations/microsoft_onedrive.html)进行操作。
