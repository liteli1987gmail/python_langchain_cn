# 黑板

>[Blackboard Learn](https://en.wikipedia.org/wiki/Blackboard_Learn)（以前称为 `Blackboard学习管理系统`）
> 是由Blackboard Inc.开发的基于web的虚拟学习环境和学习管理系统。
> 该软件具有课程管理、可定制的开放架构和可扩展的设计，可以与学生信息系统和认证协议集成。它可以安装在本地服务器上，也可以由Blackboard ASP Solutions托管，或者作为基于Amazon Web Services的软件即服务提供。
> 其主要目的是在传统面对面的课程中添加在线元素，以及开发几乎没有或没有面对面会议的完全在线课程。
