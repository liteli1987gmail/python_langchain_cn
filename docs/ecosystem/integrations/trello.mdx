# Trello

>[Trello](https://www.atlassian.com/software/trello) 是一个基于Web的项目管理和协作工具，允许个人和团队组织和跟踪任务和项目。它提供了一个被称为“看板”的可视化界面，用户可以创建列表和卡片来表示他们的任务和活动。
>TrelloLoader 允许我们从`Trello`板加载卡片。


## 安装和设置

```bash
pip install py-trello beautifulsoup4
```

请参阅[设置说明](/docs/modules/data_connection/document_loaders/integrations/trello.html)。


## 文档加载器

请参阅[使用示例](/docs/modules/data_connection/document_loaders/integrations/trello.html)。

from langchain.document_loaders import TrelloLoader
