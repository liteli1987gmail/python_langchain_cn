# Notion DB

>[Notion](https://www.notion.so/)是一个具有改进的Markdown支持的协作平台，集成了看板
>板、任务、维基和数据库。它是一个集合了记笔记、知识和数据管理，
>以及项目和任务管理的多合一工作空间。

请参阅[NotionDirectoryLoader的使用示例](/docs/modules/data_connection/document_loaders/integrations/notion.html)。


```python

```
