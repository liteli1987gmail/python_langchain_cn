# Twitter（推特）

>[Twitter（推特）](https://twitter.com/) 是一种在线社交媒体和社交网络服务。


## Installation and Setup（安装和设置）

```bash
pip install tweepy
```

我们必须使用 `Twitter API` 令牌来初始化加载器，并且我们需要设置 Twitter `用户名`。


## Document Loader（文档加载器）

查看[使用示例](/docs/modules/data_connection/document_loaders/integrations/twitter.html)。

```python
from langchain.document_loaders import TwitterTweetLoader
```
