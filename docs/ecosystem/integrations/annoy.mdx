# Annoy

> [Annoy](https://github.com/spotify/annoy)（`近似最近邻 Oh Yeah`）是一个C++库，带有Python绑定，用于搜索接近给定查询点的空间中的点。它还创建大型的只读基于文件的数据结构，将其映射到内存中，以便许多进程可以共享相同的数据。
## 安装和设置


```bash
pip install annoy
```
