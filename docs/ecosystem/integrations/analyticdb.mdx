# AnalyticDB（分析型数据库）

本页面介绍如何在 LangChain 中使用 AnalyticDB 生态系统。

### VectorStore（向量库）

存在一个围绕 AnalyticDB 的包装器，允许您将其用作向量库，
无论是用于语义搜索还是示例选择。

要导入此向量库：
```python
from langchain.vectorstores import AnalyticDB
```

有关 AnalyticDB 包装器的更详细演练，请参阅 [此笔记本](/docs/modules/data_connection/vectorstores/integrations/analyticdb.html)。
