# Metal（ 金属 ）

本页面介绍如何在 LangChain 中使用 [金属（Metal）](https://getmetal.io)。

## 什么是金属（Metal）？

金属（Metal）是一个用于生产环境的受控检索和内存平台。可以轻松将数据索引到 `Metal` 中，并在其上运行语义搜索和检索。

![Metal](/img/MetalDash.png)

## 快速入门

通过 [创建一个金属（Metal）账户](https://app.getmetal.io/signup) 开始使用金属（Metal）。

然后，您可以轻松利用 `MetalRetriever` 类开始检索数据以进行语义搜索、提示上下文等。该类接受一个 `Metal` 实例和一个参数字典，用于传递给金属（Metal）API。

```python
from langchain.retrievers import MetalRetriever
from metal_sdk.metal import Metal


metal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");
retriever = MetalRetriever(metal, params={"limit": 2})

docs = retriever.get_relevant_documents("search term")
```
