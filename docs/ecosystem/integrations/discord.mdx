# Discord（Discord）

>[Discord](https://discord.com/)是一种VoIP和即时通讯社交平台。用户可以通过语音通话、视频通话、文字消息、媒体和文件在私聊或称为“服务器”的社区中进行交流。
> 服务器是一组持久的聊天室和语音频道，可以通过邀请链接访问。
