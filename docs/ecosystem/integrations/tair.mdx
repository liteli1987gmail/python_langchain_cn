# Tair（ 天空 ）

本页面介绍如何在 LangChain 中使用 Tair 生态系统。

## 安装和设置

使用 `pip install tair` 安装 Tair Python SDK。

## 包装器

### VectorStore（ 向量存储 ）

存在一个围绕 TairVector 的包装器，允许您将其用作向量存储，
无论是用于语义搜索还是示例选择。

要导入此向量存储：

```python
from langchain.vectorstores import Tair
```

有关 Tair 包装器的更详细演示，请参阅 [此笔记本](/docs/modules/data_connection/vectorstores/integrations/tair.html)
