# Jina

本页面介绍了如何在 LangChain 中使用 Jina 生态系统。
它分为两个部分：安装和设置，然后是具体的 Jina 包装器的参考。

## 安装和设置
- 使用 `pip install jina` 安装 Python SDK
- 从 [这里](https://cloud.jina.ai/settings/tokens) 获取 Jina AI Cloud 的认证令牌，并将其设置为环境变量（`JINA_AUTH_TOKEN`）

## 包装器

### 嵌入

存在一个 Jina 嵌入包装器，您可以通过以下方式访问
```python
from langchain.embeddings import JinaEmbeddings
```
有关更详细的步骤，请参阅 [此笔记本](/docs/modules/data_connection/text_embedding/integrations/jina.html)
