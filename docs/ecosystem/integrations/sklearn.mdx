# scikit-learn

本页面介绍如何在 LangChain 中使用 scikit-learn 包。
分为两个部分：安装和设置，以及对特定 scikit-learn 包装器的引用。

## 安装和设置

- 使用 `pip install scikit-learn` 安装 Python 包

## 包装器

### VectorStore

`SKLearnVectorStore` 为最近邻实现提供了一个简单的包装器
scikit-learn 包，可以将其用作向量存储。

要导入这个向量存储：

```python
from langchain.vectorstores import SKLearnVectorStore
```

有关 SKLearnVectorStore 包装器的更详细的步骤，请参阅 [此笔记本](/docs/modules/data_connection/vectorstores/integrations/sklearn.html)。
