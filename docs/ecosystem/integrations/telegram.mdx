# Telegram

>[Telegram Messenger](https://web.telegram.org/a/) 是一款全球可访问的免费、跨平台、加密、基于云和集中式的即时通讯服务。该应用程序还提供可选的端到端加密聊天、视频通话、VoIP、文件共享和其他几个功能。
