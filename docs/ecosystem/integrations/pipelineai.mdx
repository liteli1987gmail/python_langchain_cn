# PipelineAI

本页面介绍如何在 LangChain 中使用 PipelineAI 生态系统。
它分为两个部分：安装和设置，以及特定 PipelineAI 包装器的参考。

## 安装和设置

- 使用 `pip install pipeline-ai` 安装
- 获取 Pipeline Cloud API 密钥，并将其设置为环境变量（`PIPELINE_API_KEY`）

## 包装器

### LLM

存在一个 PipelineAI LLM 包装器，您可以通过以下方式访问

```python
from langchain.llms import PipelineAI
```
