# 阿莱夫阿尔法 aleph-alpha

>[阿莱夫阿尔法](https://docs.aleph-alpha.com/)成立于2019年，致力于研究和构建强人工智能时代的基础技术。这个由国际科学家、工程师和创新者组成的团队研究、开发和部署具有变革性的人工智能，如大型语言和多模态模型，并运行欧洲最快的商业人工智能集群。

>[辉煌系列](https://docs.aleph-alpha.com/docs/introduction/luminous/)是一系列大型语言模型。

## 安装和设置

```bash
pip install aleph-alpha-client
```

您需要创建一个新的令牌。请参阅[说明](https://docs.aleph-alpha.com/docs/account/#create-a-new-token)。

```python
from getpass import getpass

ALEPH_ALPHA_API_KEY = getpass()
```
