# Chroma

> [Chroma](https://docs.trychroma.com/getting-started) 是一个用于构建具有嵌入的 AI 应用程序的数据库。

## 安装和设置

```bash
pip install chromadb
```


## VectorStore

存在一个围绕 Chroma 向量数据库的包装器，允许您将其用作向量存储，
无论是用于语义搜索还是示例选择。

```python
from langchain.vectorstores import Chroma
```

有关 Chroma 包装器的更详细说明，请参阅 [此笔记本](/docs/modules/data_connection/vectorstores/integrations/chroma.html)
