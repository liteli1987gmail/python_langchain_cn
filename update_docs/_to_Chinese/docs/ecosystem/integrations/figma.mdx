# Figma

>[Figma](https://www.figma.com/) 是一个用于界面设计的协作式Web应用。

## 安装和设置

Figma API 需要访问令牌（access token）、节点ID（node_ids）和文件密钥（file key）。

文件密钥可以从URL中获取。https://www.figma.com/file/{filekey}/sampleFilename

节点ID也可以在URL中找到。点击任何内容，查找 '?node-id={node_id}' 参数。

访问令牌的获取方法请参考[这里](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens)。
